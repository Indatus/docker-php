# PHP Docker Image

## Overview

This is an extension of the the official Docker PHP image for the purpose of added additional functionality that is ready to build.

## Versions

PHP 5.4 with Apache

* PDO_MySQL
* Mbstring
* Mcrypt
* GD
* Soap

PHP 5.5 with Apache

* PDO_MySQL
* Mbstring
* Mcrypt
* GD
* Soap
